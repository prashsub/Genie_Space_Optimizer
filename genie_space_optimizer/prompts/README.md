# Genie Space Optimizer — APX Build Prompts

A set of 8 sequential build prompts that, together, produce the complete **Genie Space Optimizer** Databricks App using [apx](https://github.com/databricks-solutions/apx).

> **How to use:** Feed each prompt to an AI assistant in order (BUILD-01 → BUILD-08). Each prompt is self-contained with context, references, acceptance criteria, and dependency information. The assistant should read the referenced architecture docs, then implement the specified modules.

---

## Execution Order & Dependencies

```
BUILD-01: Foundation (common/)
    ├──→ BUILD-02: Delta State Machine
    ├──→ BUILD-03: Evaluation Engine
    └──→ BUILD-04: Patch DSL, Optimizer, Applier
              │
              ▼
         BUILD-05: Optimization Harness
              │
              ▼
         BUILD-06: Jobs + FastAPI Backend (APX)
              │
              ▼
         BUILD-07: React Frontend (APX)
              │
              ▼
         BUILD-08: Deployment & Testing
```

BUILD-02, BUILD-03, and BUILD-04 can run **in parallel** after BUILD-01 completes.

## APX Conventions (applies to all prompts)

| Area | Convention |
|------|-----------|
| Package root | `src/genie_space_optimizer/` |
| Python manager | `uv` (not pip) |
| JS manager | `bun` (not npm) |
| Backend auth | `Dependencies.Client` / `Dependencies.UserClient` |
| Config access | `Dependencies.Config` (Pydantic `AppConfig`) |
| Router creation | `create_router()` (not `APIRouter()`) |
| API prefix | `api-prefix = "/api/genie"` in `pyproject.toml` |
| Frontend routing | TanStack Router (file-based in `ui/routes/`) |
| UI components | shadcn/ui via `apx components add` |
| API hooks | Auto-generated from OpenAPI (no manual `fetch()`) |
| Build | `apx build` → Python wheel with embedded frontend |
| Deploy | `databricks bundle deploy -p <target>` |
| Dev server | `apx dev start` (all-in-one) |

## Prerequisites

Before starting BUILD-01:

1. **Install APX:** `curl -fsSL https://databricks-solutions.github.io/apx/install.sh | sh`
2. **Initialize project:** `apx init genie-space-optimizer --addons=ui,sidebar,sql,cursor --profile dev`
3. **Configure:** Update `pyproject.toml` and `databricks.yml` per `APX-BUILD-GUIDE.md` Steps 2a-2c
4. **Verify:** `apx dev start` runs without errors
